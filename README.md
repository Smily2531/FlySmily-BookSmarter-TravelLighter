## 📌 Project Overview
FlySmily addresses the complexities of flight management by providing a multi-role ecosystem:
- **Customers**: Effortless searching and booking.
- **Admins**: Full control over users, flights, and system health.
- **Flight Operators**: Real-time status management and passenger tracking.

---

## ✅ Core Features

### 👤 Customer Experience
- **Smart Search**: Find flights by origin, destination, and dates.
- **Seamless Booking**: Real-time seat selection and booking confirmation.
- **Personalized Dashboard**: View and manage upcoming trips.
- **Secure Authentication**: JWT-based login and registration.

### 🛠️ Administrative Control
- **User Management**: Oversee customer and operator accounts.
- **Inventory Control**: Add, update, or remove flight schedules.
- **System Analytics**: View all bookings and operational data.

### ✈️ Operational Efficiency
- **Flight Status Updates**: Real-time tracking of delays or cancellations.
- **Passenger Manifest**: Access passenger lists for assigned flights.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- MongoDB (Running locally or on Atlas)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Smily2531/FlySmily-BookSmarter-TravelLighter.git
   cd FlySmily-BookSmarter-TravelLighter
   ```

2. **Setup Server:**
   ```bash
   cd server
   npm install
   # Update MongoDB URI in index.js if necessary
   npm start
   ```

3. **Setup Client:**
   ```bash
   cd ../client
   npm install
   npm start
   ```
